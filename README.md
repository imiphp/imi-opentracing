## 介绍

此项目是 imi 框架的调用链路追踪（OpenTracing）组件。

**支持的功能：**

* [x] Zipkin

* [x] Jaeger

## 安装

`composer require imiphp/imi-opentracing:~2.1.0`

## 使用说明
